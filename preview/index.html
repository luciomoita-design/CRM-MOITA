Como explorar rapidamente

Estes mockups estáticos representam os fluxos principais do Moita CRM e podem ser abertos em qualquer navegador. Cada tela foi montada com os mesmos componentes e diretrizes de UI utilizadas no app Next.js, permitindo validar o layout, copy e interações antes de subir o ambiente completo.

  1. 1 Acesse os cards abaixo para abrir cada visão em uma nova aba.
  2. 2 Use o modo claro/escuro embutido nos mockups para conferir as duas variações de tema.
  3. 3 Para uma experiência completa, siga o README e execute docker compose up.

Quer testar com dados reais?

Siga o passo a passo do README para subir toda a stack com Docker (PostgreSQL, Redis, BullMQ, Next.js e coletor OpenTelemetry). O seed já inclui organização demo, pipeline padrão e leads para explorar rapidamente.

$ cp .env.example .env $ docker compose up --build $ open http://localhost:3000
